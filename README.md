# allergylocate.org
Free, crowdsourced database of verified allergen information from US restaurants.

**Live site:** https://allergylocate.org

## What it does
- Indexes publicly available allergen PDFs and pages from 100+ food establishments
- Fully searchable and filterable by chain, independent, and NYC locations
- Accepts community submissions of new restaurants

## Policy context
This project supports NJ Senate Bill S3394, which would require restaurants to
publicly disclose allergen information. This database demonstrates that the
infrastructure for allergen transparency already exists.
